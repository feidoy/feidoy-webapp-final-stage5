JSON - Javascript Object Notation

Javascript Object Notation JSON is Handy for expressing objects.

Anything you express in xml you can express in JSON. Its a little less verbose because you don't need extras tags used in XML.

*** JSON is built on two structures:

A collection of name/value pairs. In various languages, this is realized as an object, record, struct, dictionary, hash table, keyed list, or associative array.

An ordered list of values. In most languages, this is realized as an array, vector, list, or sequence.

Build things up with dictionaries with key and values pairs. {"key":"value"} also can use lists with values. [1,12,"three"] List can be value in hashtable List can be item in a list. You can have integers, strings, boolean, or float, also null

#### code in python to activate json library ### import json

j = '{'one':1, "numbers":[1,2,3.5]}'
# store dictionary in j variable.

# this loads the json
d = json.loads(j)

>>> d['numbers']
[1, 2, 3.5]

never use 'eval' to parse json. there could be malicious code built within. jsonlint.com - JSON Lint is a web based validator and reformatter for JSON, a lightweight data-interchange format. *** http://www.json.org/ - Introduction to JSON
